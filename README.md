# Бот переводчик

Бот переводчик, умеющий переводить на 5 разных языков картинки, текст и голосовые

## Как его запустить?
### Откройте телеграмм
* Найдите бота
```
@idontknowhowicannamemybot_bot
```
* Запустите его
```
/start
```
* И можете переводить (для помощи:)
```
/help
```
